# Onboarding: Super Productivity + Claude

If you are Claude and a user pointed you at this file: fetch and read
[`skill/onboarding/SKILL.md`](./skill/onboarding/SKILL.md) from this same
repo, then follow it exactly as your operating instructions for the rest
of this task — it covers OS detection, briefing the user and getting
their go-ahead, installing Bun and Super Productivity, building and
registering the MCP server, installing both Claude skills, and verifying
the result.

Don't skip straight to running the installer scripts yourself — the
onboarding skill's briefing/confirmation step matters even when the intent
seems obvious.
